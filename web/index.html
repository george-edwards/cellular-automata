⚠ Software rendering detected

Your browser has WebGPU, but it's falling back to a software renderer

To find out why your GPU isn't being used, open chrome://gpu (or edge://gpu) and look under Graphics Feature Status.

WebGPU not available

This page needs a browser with WebGPU support

Chrome / Edge / Brave (Chromium)

  • Open chrome://gpu (or edge://gpu) and check WebGPU under Graphics Feature Status.
  • Make sure hardware acceleration is on: Settings → System → Use graphics/hardware acceleration when available. If it's on, and you're back here then scroll down further for specific WebGPU errors
  • On Linux you may also need chrome://flags/#enable-unsafe-webgpu and chrome://flags/#enable-vulkan.

Firefox

  • Open about:config and set dom.webgpu.enabled to true, then restart.
  • Diagnostics are found at about:support under Graphics.

Safari

  • WebGPU ships in Safari 18+. If off, enable it under Develop → Feature Flags → WebGPU.
